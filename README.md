# KIMCARD

A modern cyber dashboard UI built with Next.js, React, Tailwind CSS, and Framer Motion.

## Features
- Glassmorphism / Liquid Glass UI
- Dark neon cyber theme
- Responsive dashboard layout
- Animated transitions
- Demo-only sections (no illegal logic)
- Component-based structure

## Tech Stack
- Next.js (App Router)
- React
- Tailwind CSS
- Framer Motion
- Recharts
- Lucide React Icons

## Getting Started

1. Install dependencies:
   ```
   npm install
   ```
2. Run the development server:
   ```
   npm run dev
   ```
3. Open [http://localhost:3000](http://localhost:3000) in your browser.

This project is for UI/UX demonstration only. No illegal features or payment abuse are implemented.
